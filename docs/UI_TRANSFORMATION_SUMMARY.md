## 📐 Design System

### Typography:
- **Body**: Inter (300-800 weights)
- **Headings**: Playfair Display (600-800 weights)

### Spacing:
- **Small**: 8px, 12px, 16px
- **Medium**: 20px, 24px, 30px
- **Large**: 40px, 50px, 60px

### Border Radius:
- **Small**: 8-12px
- **Medium**: 16-20px
